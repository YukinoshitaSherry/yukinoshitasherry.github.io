## 托福简介

### 题型
2023改版新托福
顺序：阅读→听力→口语→写作

| 考试部分 | 考试内容         | 题目数量         | 预估时长 |
|----------|------------------|------------------|----------|
| 阅读 30 分 | 2 篇阅读文章     | 每篇 10 道题目   | 35 分钟  |
| 听力 30 分 | 2 段对话         | 每篇 5 道题目    | 36 分钟  |
|          | 3 段讲座         | 每篇 6 道题目    |          |
| 口语 30 分 | 1 道独立任务     | 4 道题目         | 16 分钟  |
|          | 3 道综合任务     |                  |          |
| 写作 30 分 | 1 道综合任务     | 2 道题目         | 29 分钟  |
|          | 1 道讨论写作     |                  |          |
| 总分     | 120 分           |                  | 总时长在 2 小时以内 |

### 考试流程
#### 报名
网址：[https://toefl.neea.cn/](https://toefl.neea.cn/)
记得提早抢考位。
看清楚转考与取消的时间与金钱损失政策。

#### 前往考场
提前打印确认信，带好确认信和身份证，不要走错考点，比确认信要求的时间再早一点到达。
之后听指挥即可。
大致是入口签到——领承诺书——存包——安检——填写承诺书——检录进考场，拍照之类的——坐着别动等工作人员给你发三张纸，两只铅笔，并解锁你电脑的考试

#### 考试
一堆信息没啥用，点continue即可——试音，调整设备【晚一点结束试音，后面会讲】——阅读——听力——口语——写作——结束，显示阅读与听力得分——点report，不要点cancel，否则就是自己取消了自己成绩——举手示意老师，可以带走一支笔，出考场

#### 查分
官网：[https://toefl.neea.cn/](https://toefl.neea.cn/)
微信小程序：托福官方考试平台 (可以提前查分，准的。可能考完3~5天会出)

托福的听力、阅读存在curve机制，即根据本场考试整体水平与题目难度调节某一套卷子的分数，可能会比考完看到的分数+—2分，据说发生概率是10%。笔者最后一次考试阅读被curve高了1分，考场看到29，查分30。
若成绩显示异常，不要慌张。若被Hold，耐心等待即可，会出的比较晚，分数可能变低（也不一定)。若被取消，给ETS官方打电话/发邮件说明情况，小红书有不少案例帖子可搜索。
若觉得分数不合理可以上官网，多花点钱申诉，分数可能变高/变低/不变。


## 注意事项
1.提前打印确认信、考前准备好确认信、身份证，一定要记得带。
2.提前去考点踩点，以确认信地址为准，不要导航导错，浙江考试院导航结果可能有两家，不要走错！
3.注意确认信时间，提前到，过了时间真的不能入内。早上场一般是8:45正式开考，一定要在8:30之前进去！进去以后存包安检填写表等等会花费15分钟，个人建议总共一小时前到考场，门口再看会儿注意事项/作文口语再进去。
4.进考场后按指示行动即可，考场不能带任何东西(包括水、非智能手表、纸巾)进入，进去候考时没有水喝(口语是第三个part，注意自己控制嗓子条件)，但还可以上厕所，会提供纸。
5.浙江省考试院的半包封闭式电脑桌私密性稍好于浙江大学紫金港校区东六的圆桌，键盘个人感觉也是考试院的更舒适，隔音两个都很差。
6.会发铅笔两支，是那种可以取下头部插入尾部的，质量还行，建议平时就用类似笔练习。会发草稿纸三张，一般是彩色的，用完可以举手示意更换，但是会一次性更换全部三张。合理的分区+字写小一点，一般三张够用的。最后可以带走一支笔。
7.开始前会有试音，可以自己控制一下点击按钮continue的进度，不要太早完成试音进入阅读部分的答题。这个宁晚不早，否则早早结束试音进阅读部分，同考场的其他同学还在试音，他们的说话声可能干扰你专心做阅读。不会分心者可以忽略此条。
8.无论试音还是口语都不要说话太小声！如果试音通过那么就代表音量可以，试音只有一次，记住你大概的说话声响。正式考口语时，语气可以适当drama有起伏一点。而听力的音量全程可调节，点击右上角volume按钮即可。
9.每个考生按自己的节奏答题，进度有快慢，可能你还在做听力的时候就有人开始说口语了，尽量不要被影响，平时练习听力时可以在稍嘈杂的环境插上耳机做题(比如上水课的时候)。同一场考试有好几套不同的题，所以企图利用做题时间差先听别人口语的主题以提前构思的trick不太现实，不如专注自己的部分。
10.注意右上角倒计时，控制好时间。听力阅读一道题想不出来不要卡太久。尤其阅读，最后还可以回去看。听力不能回去看了，实在不行差不多蒙一个往下，损失这一个题总比最后没来得及做完好。

### 背单词
- 单词书
    - 王玉梅(方富民老师托福阅读课指定单词书)
    - 词以类记(学科词汇积累，比较难)
    - 俞敏洪 红宝书(这本也行)
- 语料积累实用网站：
    - 中国日报英语点津 [http://language.chinadaily.com.cn/](http://language.chinadaily.com.cn/) —可以查找中国特色的一些词的专业英语翻译
    - 语料库合集 [https://www.english-corpora.org/](https://www.english-corpora.org/) —主要是北美英语语料库（COCA），看里面的frequency看一个表达是口语用词（spoken）还是书面用词（academic）
    - 搭配查询 [https://ozdic.com/](https://ozdic.com/) —查找一个搭配是否正确（比如输入knowledge，可以看到跟它搭配最多的形容词，动词等）
    - 同义词词典 [https://www.merriam-webster.com/thesaurus](https://www.merriam-webster.com/thesaurus) —查同义替换的网站，要注意将查到的词进行二次确认（因为一个单词不同的意思对应的同义词也不一样），同时要注意单词不同的词性对应的同义词的形式。也有手机app,就叫Pocket Thesaurus。
