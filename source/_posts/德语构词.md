德语属于典型的**综合型语言**：许多概念用一个词块表达，尤其名词常通过**复合**（Komposition）与**派生**（Derivation）拉长。掌握构词规律能显著减轻记忆负担，并在阅读时拆词猜义。本篇笔记按 结构类型 → 规则 → 高频缀合 → 与英语对照 组织，便于当作工具笔记查阅。

附一个AI辅助解释生词的prompt (from Woo老师)：
> 你是一名经验丰富的德语教师，接下来我会每次给你一个德语单词，我需要你向我提供单词的基本信息(含义和词性)、记忆方式(构词法, 英语对应词, 相近的词汇)，便于我学习和记忆这个单词。


<br>

## 构词类型总览

| 类型 | 德语术语 | 说明 | 典型结果 |
| :--- | :--- | :--- | :--- |
| 复合 | Komposition / Zusammensetzung | 两个及以上词干直接拼合，中间可有连接成分 | *Hand*（手）+*Schuh*（鞋）→ *Handschuh*（手套） |
| 派生 | Derivation / Ableitung | 词干 + 前缀/后缀改变词类或语义 | *lesen*（读）→ *Leser*（读者），*lesbar*（可读的） |
| 缩略 | Abkürzung / Kurzwort | 首字母、截断、拼合 | *Kfz*（机动车），*Uni*（大学） |
| 外来构词 | Lehnwortbildung | 拉丁/希腊/法语词缀与词干组合 | *Biologie*（生物学），*telefonieren*（打电话） |

> [!INFO]+ 与英语的主要差异
> - 德语名词复合**无空格**，书写上一词到底；英语常保留空格或用连字符（*traffic light* vs *Ampel* (红绿灯)。虽为单词，但德语更长复合更常见。
> - 德语**名词首字母大写**，复合词中每个成分在词典里若本为名词，概念上仍可分析；动词/形容词参与复合时仍遵循整体词类规则。
> - **性别**由**最后一个名词成分**决定（见下文）。

<br>

## 复合名词 Komposita

### 基本规则

1. **语义**：多为(限定关系)——前一个成分修饰后一个（中心语在后，与中文(棒球帽)语序类似）。
2. **性**：**整个词的性 = 最后一个名词词干的性**。  
   - *die Hand*（手）+*der Schuh*（鞋）→ **der** Handschuh(手套)（性随 *Schuh*(鞋)）
3. **复数**：通常由**最后一个名词成分**的复数规则决定：*die Handschuhe*(手套（复数）)。
4. **重音**：多数情况下**落在第一个成分**（除非有不可分前缀等特殊结构，复合名词一般首重音）。

> [!EXAMPLE]+ 例1：中心语在尾
> - *Universität*（大学）+*Bibliothek*（图书馆）→ *die Universitätsbibliothek*（大学图书馆）
> - *Kraft*（力；动力）+*Fahrzeug*（交通工具）→ *das Kraftfahrzeug*（机动车）

<br>

### 连接成分 Fugenelemente / Fugenlaute

复合时中间常插入 **-s-、-es-、-n-、-en-** 等，并非随意：与语音和谐、历史词形、书面固定写法有关。初学可**整词记忆**，进阶再归纳。

| 连接形式 | 常见出现环境 | 举例 |
| :--- | :--- | :--- |
| **-s-** | 极常见，尤其与辅音起首的后成分 | *Liebe*（爱）+*s*+*Beweis*（证明）→ *der Liebesbeweis*(爱的证明)；*Jahreszeit*(季节) |
| **-es-** | 部分固定复合 | *Tag*（日）+*es*+*Zeitung*（报纸）→ *die Tageszeitung*(日报) |
| **连接 -e-** | 名词词干加 *e* 再接下一成分 | *Schwein*（猪）+*e*+*fleisch*（肉）→ *das Schweinefleisch*(猪肉) |
| **-n- / -en-** | 部分复合中元音词干加连接 -n- | *Woche*（周）+*n*+*Ende*（末尾）→ *das Wochenende*(周末) |
| **零连接** | 两成分直接拼接 | *Hand*（手）+*Schuh*（鞋）→ *Handschuh*(手套) |

> [!INFO]+ 学习策略
> - 词典里给出的复合词形式即为规范写法；写作时不要自造连接音。
> - 阅读时若见到陌生长词，**从右向左**找最后一个完整名词词干，往往先猜出(大类)。

<br>

### 形容词/动词参与复合

- **形容词 + 名词**：*Groß*（大的）+*Stadt*（城市）→ *die Großstadt*(大城市)。
- **动词词干 + 名词**（表示动作/工具/结果）：*Schreib-*（写）+*Maschine*（机器）→ *die Schreibmaschine*(打字机)（历史词，字面(书写机器)）。
- 此类词的性和复数仍由**名词部分**决定。

<br>

## 派生：动词前缀与可分性

德语动词前缀分**可分**与**不可分**，影响**第二分词（过去分词）里 *ge-* 是否出现及位置**、**句中动词框**等；构词时需一并记忆。

### 常见不可分前缀（重音在词干上）

| 前缀 | 大致语义倾向 | 举例 |
| :--- | :--- | :--- |
| **be-** | 及物化、使涉及 | *suchen*(寻找)→ *besuchen*(拜访；参观) |
| **er-** | 完成、获得、开始 | *kennen*(认识)→ *erkennen*(认出) |
| **ver-** | 偏离、错误、加强、失去 | *kaufen*(买)→ *verkaufen*(卖；出售) |
| **ent-** | 去除、离开 | *decken*(遮盖)→ *entdecken*(发现) |
| **emp-**（在 m/p 前） | 与 ent- 相关 | *empfehlen*(推荐) |
| **miss-** | 错误 | *verstehen*(理解)→ *missverstehen*(误解) |
| **zer-** | 碎裂、彻底 | *reißen*(撕)→ *zerreißen*(撕碎) |

> [!INFO]+ *ent-* 前缀与 *Entschuldigung* 
>
> - *die Schuld*「罪责；亏欠；过错」→ *schuldig*「有罪的；该负责的」→ 动词 *entschuldigen*「为…开脱；道歉；请求原谅」→ 名词 *die Entschuldigung*「道歉（的话）；借口；原谅」。
> 为表中 *ent-* 同类：**去除、使脱离**。在 *entschuldigen* 里可理解为「使（某人）**脱离**过错/罪责负担」→ 引申为道歉、辩解。
> - 名词：**Ent-** + **schuld** + **-igung** → *Entschuldigung*（*schuld* 为词根「罪责」）。  
> - 后缀 **-igung**：对应动词 *entschuldigen*（词尾 *-igen*）的名词化，与 *-ung* 平行的一类（同类动词如 *reinigen* → *die Reinigung*）。**不是** *Ent* + *Schuldigung*（不存在 *Schuldigung* 这一独立成分）。
>
> **用法提示**  
> - 反身：*sich bei jemandem für etwas entschuldigen*「因某事向某人道歉」。  
> - 感叹/应酬语：*Entschuldigung!*「对不起；打扰一下」；口语也常说 *Tschuldigung*（缩略）。

<br>


### 可分前缀（重音在前缀上）

可分前缀多带**空间、方向、体貌**色彩，但许多已**虚化**，同一前缀在不同动词里义差别大；下表只列**常见倾向**，具体仍以词典义项为准。

| 前缀 | 大致语义倾向 | 举例 | 句中位置提示 |
| :--- | :--- | :--- | :--- |
| **an-** | 朝向、贴靠、接触；与某些动词表(着手、开始) | *anfangen*(开始) | *Ich fange an.*(我开始。)（*an-* 在句尾） |
| **auf-** | 向上、打开、完全；与站立类动词常表(起身) | *aufstehen*(起床；起立) | *Ich stehe auf.*(我起床。)（*auf-* 在句尾） |
| **mit-** | 一起、随同、附带 | *mitkommen*(同来；跟上) | *Kommst du mit?*(你一起来吗？)（*mit-* 在句尾） |
| **zu-** | 朝向、闭合；与 *hören/sehen* 等常表(朝…方向去听/看) | *zuhören*(倾听) | *Ich höre zu.*(我在听。)（*zu-* 在句尾） |
| **ab-** | 离开、脱离、卸下、减少 | *abfahren*(发车；驶离) | *Der Zug fährt ab.*(火车开走了。)（*ab-* 在句尾） |
| **ein-** | 进入、装入、纳入 | *einkaufen*(购物) | *Ich kaufe heute ein.*(我今天去购物。)（*ein-* 在句尾） |
| **aus-** | 向外、尽、熄 | *ausgehen*(外出；（灯）熄灭) | *Wir gehen aus.*(我们外出。)（*aus-* 在句尾） |
| **vor-** | 向前、预先、在…前展示 | *vorschlagen*(建议) | *Ich schlage das vor.*(我提议这个。)（*vor-* 在句尾） |

> [!INFO]+ 与不可分表对照
> 上表(大致语义倾向)写法与上文**不可分前缀**表一致，便于对照；但可分前缀**更依赖整词记忆**，勿把某一前缀只对应一个中文义。
>
> 另有一批高频可分前缀，表义同样多为方向/体貌：**her-**（朝说话者一侧来）、**hin-**（朝远离说话者一侧去）、**zurück-**（返回）、**los-**（动身、开始）、**weg-**（离开）、**bei-**（附带，如 *beibringen* 教学）等，具体搭配须查词典。

> [!EXAMPLE]+ 例2：可分与不可分同形
> - *übersetzen*（不可分）(翻译)；*übersetzen*（可分）≈ *hinüber-setzen*(渡过；摆到…对面)——重音与分词形式不同，需查词典确认。

<br>

> [!INFO]+ *auferstehen*（旧拼 *auferstehn*, 马勒二交响曲的那个）
>
> **先分清两个不同的动词**  
> - *aufstehen*：**可分**。前缀 *auf-*「向上」+*stehen*「站立」→「起床；起立」。陈述句前缀在句尾：*Ich stehe um sieben auf.*「我七点起床。」第二分词：*aufgestanden*（*ge-* 插在 *auf-* 与 *steh-* 之间：*aufge-* + *standen*）。  
> - *auferstehen*：**不可分**。整词义为「（从死亡中）复活；再起」，多用于**宗教、文学、固定说法**，不能按「*auf-* 甩句尾」理解。
>
> **auferstehen** 的形态与来源（为何看起来像 *auf-*） 
> 历史层次上常分析为 **auf-** + **er-** + **stehen**：*auf-* 表「向上」，*er-* 在此类古语复合里常带「再、重新、完全」一类色彩（与可产的 *erkennen* 之 *er-* 不完全相同），合流后写作固定词干 **aufer-** + *stehen*。现代德语里应把 **aufer-** 当作**整体前缀**记忆，**勿**自行拆成可分的 *auf … erstehen*。
>
> **语法与分词**  
> - 陈述句：*Er ist auferstanden.*「他复活了。」（前缀**不**甩到句尾。）  
> - 第二分词：*auferstanden*（不可分动词：**无** *ge-*）。  
> - 名词：*die Auferstehung*「复活」（尤指耶稣复活节语境）。
>
> **拼写说明**  
> 标准动词为 *auferstehen*；旧式/诗歌里偶见 *auferstehn*（与口语 *stehn* 同系非完整规范形式），教材与词典仍以 *-stehen* 为准。

## 名词派生后缀（抽象名词与施事）

下列后缀多将动词/形容词名词化，**性多为阴性或中性**，但有例外，须死记典型词。

| 后缀 | 常见性 | 含义/功能 | 举例 |
| :--- | :--- | :--- | :--- |
| **-ung** | 多为 die | 行为、结果、机构 | *die Bildung*(教育；形成) |
| **-heit / -keit** | die | 性质、状态 | *die Freiheit*(自由)，*die Möglichkeit*(可能性) |
| **-nis** | 多为 das/die | 结果、状态 | *das Ergebnis*(结果)，*die Kenntnis*(知识；了解) |
| **-schaft** | die | 集合、领域 | *die Freundschaft*(友谊) |
| **-tum** | das | 状态、集合 | *das Christentum*(基督教) |
| **-er** | der | 施事、工具、来源 | *der Leser*(读者)，*der Berliner*(柏林人；（小写时）柏林果酱甜甜圈) |
| **-ling** | der | 带某种特征的人/物 | *der Schützling*(受保护者；受监护人) |
| **-in** | die | 女性职业/身份 | *die Lehrerin*(女教师) |

> [!INFO]+ *-ung* 与动词的关系
> 许多动词去掉 -en 加 -ung：*bedeuten*(意味着)→ *die Bedeutung*(意义)。但词干可能变音或拼写调整（*entscheiden*(决定)→ *die Entscheidung*(决定)），不能机械类推。

<br>

### 指小后缀与 **ge-** 名词化

| 手段 | 说明 | 举例 |
| :--- | :--- | :--- |
| **-chen / -lein** | 指小、亲昵或固定中性名 | *das Mädchen*(女孩)，*das Kindlein*(小家伙；幼儿)（词干常变音：*a*→*ä*） |
| **Ge- + 动词词干** | 集合或过程名词，多为中性 | *das Gebäude*(建筑物)，*das Gerede*(空谈；唠叨) |

否定前缀 **un-**（与形容词/分词）：*unmöglich*(不可能的)，*unbekannt*(未知的)；拉丁 **in-/ir-**（在部分词中）：*indirekt*(间接的)，*irregulär*(不规则的)（外来词拼写需整词记忆）。

<br>

## 形容词与副词派生后缀

| 后缀 | 功能 | 举例 |
| :--- | :--- | :--- |
| **-lich** | 具有…性质 | *freundlich*(友好的) |
| **-isch** | 与…相关、学派、语言 | *chemisch*(化学的)，*englisch*(英语的；英国的) |
| **-bar** | 可…的 | *lesbar*(可读易懂的)，*verständlich*(可理解的) |
| **-los** | 无… | *hoffnungslos*(绝望的) |
| **-voll** | 充满… | *sinnvoll*(有意义的) |
| **-sam** | 易于…、倾向… | *mühsam*(费力的；麻烦的) |
| **-haft** | 带有…特征 | *mächtig*(强大的)（与 *-haft* 不同源），但 *fehlerhaft*(有错误的)属此类 |
| **-reich / -arm** | 多/少 | *kalorienreich*(高热量的) |

**副词**：许多形容词加 **-erweise** 构成方式副词：*normalerweise*(通常)；或直接用形容词作副词（无 -ly）：*gut fahren*(车开得好)。

<br>

## 高频拉丁/希腊词缀

阅读科技、医学、社科文本时常见，可与英语对照记忆。

| 成分 | 含义示例 | 德语例词 |
| :--- | :--- | :--- |
| **Bio-** | 生命 | *die Biologie*(生物学) |
| **Geo-** | 地球 | *die Geologie*(地质学) |
| **Graph** | 写、记录 | *die Grafik*(图形；平面设计) |
| **Logie** | 学科 | *die Psychologie*(心理学) |
| **Phobie** | 恐惧 | *die Agoraphobie*(广场恐惧症) |
| **Tele-** | 远 | *das Telefon*(电话) |

英语与德语拼写常极接近，注意德语名词**首字母大写**。

<br>

## 与英语易混与拼写提示

| 现象 | 说明 |
| :--- | :--- |
| **假朋友** | 构词相似但义不同：*bekommen*(得到；收到)≠ 英语 *become*(变成)；需整词记忆。 |
| **c/k/z** | 德语本土词多用 *k*、*z*（读 /ts/）；*C* 多出现于外来词。 |
| **长复合词** | 不要按英语节奏断词；朗读时按成分边界拆分重音（通常首成分重读）。 |

<br>

## 总结

【每学一个新复合词，标注**中心语**与**连接成分**！】

1. 复合名词的性、复数看**最后一项名词**。
2. 动词前缀分清**可分/不可分**，影响句框与第二分词形式（可分：*mitkommen* → *mitgekommen*(一起来过)；不可分：*besuchen* → *besucht*(拜访过)）。
3. 抽象名词多 **-ung / -heit / -keit**，但词干变化需查词典。
4. 连接音 **-s-** 等随固定写法，勿自造。
5. 长词阅读：**从右向左**先锁定中心名词。

> [!EXAMPLE]+ 例3：拆解练习
> - *die Bundesrepublik Deutschland*(德意志联邦共和国)：*Bund*(联邦)+*es*（连接音）+*Republik*(共和国)；国名专有，整体阴性随 *Republik*(共和国)。
> - *das Kraftfahrzeug*(机动车)：*Kraft*(力；动力)+*Fahrzeug*(交通工具)→ 中性随 *Fahrzeug*(交通工具)。
