### 纯正弦波的不和谐

- 一个声音，由基频(最低的正弦波)决定音高，由更高频的泛音列决定音色。
- 除了音色，泛音还能决定音阶与和弦。
- 一个声音可以通过傅里叶变换分成不同的正弦波(见3b1b的相关视频)。
- 两个正弦波频率相同时无不和谐，其中一个不变，一个不断升高过程中，不和谐先急剧上升再逐渐下降，但不会回到0。另一个不断降低同理。即会存在一个相近的不和谐区。

<img src="https://raw.githubusercontent.com/YukinoshitaSherry/qycf_picbed/main/img/20260412233502939.png"/>

#### 原因

- **拍频**：两个频率几乎相同的正弦波在相长干涉与相消干涉之间交替，它们的振幅产生调制。频率越接近，它们回到相同相的时间越长，拍频持续越长。频率相差越大则相反，它们进出同相的速度越快，拍频越快。


> [!INFO]+ 解释
> **相长干涉**≈ 波峰对齐、声音变强；**相消干涉**≈ 峰谷相对、声音变弱；**振幅调制**≈ 音量被调制成忽大忽小；**同相**≈ 两列波又对齐了。

<br>


- **人耳的不完美分辨率**:由于耳蜗的结构与其中的液体和毛细胞的物理特性，单一频率的外部频率不仅会刺激对应该频率的听觉神经纤维，还会刺激稍高和稍低频率的纤维。如果两个频率太近，耳朵无法准确分辨之为两种不同的声音。我们听到的是单一，不和谐的音符；直到频率间隔足够远，让耳朵能把它们分辨为两个独立的声音。(戴耳机，左播放440Hz，右播放450Hz，可以分辨出两个相近的不同音高；但是摘下耳机，听起来像一个不和谐音。)

<br>


### 带有泛音的不和谐

- 当谈及两组声音的不和谐，不仅是两个基频正弦波之间的不和谐，还包括一个基频正弦波和另一个音的所有泛音列之间的不和谐，以及所有泛音列之间的不和谐。

#### 参考代码

- <a href="https://github.com/aatishb/dissonance">视频作者给的参考代码仓库</a>

下面这段与上面的2D的图同类：**固定参考音 A4 = 440 Hz（带谐波列）**，横轴为**另一音的基频 x**，纵轴为 Plomp–Levelt 型**感知不和谐度**之和。代码在相邻频率差很小时会给出尖峰，在 x = 440 处两组音完全重合故为深谷（与仅两列纯正弦时的光滑曲线不同，这里要对**所有谐波对**求和）。

**字母和图上的位置怎么对应（读图）：**

- **横轴**：就是代码里的 **`x`**（以及画图用的数组 **`xs`**）——对比音的**基频**，单位 Hz。你从曲线左端扫到右端，就是在让 **`x`** 从小到大变。
- **纵轴**：**`my_sound_ds(x)`** 的返回值（数组 **`ys`**）——在**某一个横坐标 `x`** 上，曲线**往上有多高**，就是总不和谐度有多大（越高越“糙”）。
- **`f_ref`**：它告诉程序参考音钉在 **440 Hz**，所以整条曲线在 **横轴 ≈ 440** 处会出现**最深的谷**（两串谐波完全重合）。
- **`f1`、`f2`**：它们是**每一对**谐波分量（参考第 *i* 个谐波 vs 对比第 *j* 个谐波）的频率；**25 对**全部加起来，才得到**某一个 `x` 处**纵轴上的那**一个**点。
- **`l1`、`l2` / `amps`**：对应**振幅**（响度权重）。决定**每一对**能贡献多高（越弱的谐波，这一对能顶上去的上限越低，因为公式里有 **`min(l1,l2)`**）。
- **`s`、`p`**：它们把“这一对 `f1` 和 `f2` 差多少 Hz”变成无量纲的 `p`，再代入双指数；`p` 落在中间某段时这一对不和谐最大——许多对一起在 440Hz 两侧叠加，就出现尖峰。

```python
import numpy as np
import matplotlib.pyplot as plt


# ---------- Plomp–Levelt：单对分量 (f1,l1) 与 (f2,l2) 的局部感知不和谐度 ----------
# f1,f2：两列正弦分量的频率 (Hz)。l1,l2：对应振幅（响度权重）。
# p：无量纲归一化频差；s：把 Δf(Hz) 映射到与临界带尺度相容的轴上的尺度因子（把赫兹差变成“相对临界带的”无量纲步长）。
# 核函数 d(p)=exp(-b1*p)-exp(-b2*p)：p≈0 时为 0（同频无差）；p 中等时不和谐程度最大；p 很大时 →0（感知上可分离）。
def dissonance(f1, f2, l1, l2):
    # x_scale 与 s1,s2,b1,b2：Sethares/Plomp–Levelt 文献中的经验常数，拟合心理声学实验曲线
    x_scale = 0.24  # 与分母一起决定 s 的整体尺度；避免与下面“待测基频”变量名 x 混淆
    s1, s2 = 0.0207, 18.96  # 出现在 s 的分母：s = x_scale/(s1*fmin+s2)，与临界带宽度随 fmin 变化配套
    b1, b2 = 3.51, 5.75  # 双指数衰减率，控制 d(p) 从 0 升起再回落的形状（何处最“不和谐”）
    fmin, fmax = min(f1, f2), max(f1, f2)  # 用较低频 fmin 归一化，使同样 Δf 在低频/高频上权重不同
    s = x_scale / (s1 * fmin + s2)
    p = s * (fmax - fmin)  # p = s * Δf，代入 d(p)；min(l1,l2) 表示贡献受较弱分量限制
    return min(l1, l2) * (np.exp(-b1 * p) - np.exp(-b2 * p))

# ---------- 参考声：基频 f_ref，整数倍谐波 + 递减振幅（近似弦/管音色）----------
# ---------- 对比声：基频为自变量 x，谐波序号与相对振幅与参考声相同 ----------
# 对两组谐波 (i,j) 各算一对 (f_ref*h_i, x*h_j) 的 dissonance 并求和 → 总不和谐度。
def my_sound_ds(x):
    f_ref = 440.0  # 固定参考音 A4
    harmonics = [1, 2, 3, 4, 5]  # 第 n 项 = n 次谐波：频率 = 基频 × n
    amps = [1.0, 0.4, 0.3, 0.2, 0.1]  # 各次谐波相对振幅；下标与 harmonics 一一对应
    total = 0.0
    for i in range(5):
        for j in range(5):
            total += dissonance(
                f_ref * harmonics[i],  # 参考声第 i 个谐波频率
                x * harmonics[j],  # 对比声第 j 个谐波频率（x 为待扫的基频）
                amps[i],
                amps[j],
            )
    return total

# 横轴 0～1000 Hz：x≈440 深谷（同音重合）；两侧尖峰（近频快拍→不和谐程度）；
# 220、660、880 等处浅谷（谐波与参考列对齐，不和谐度局部降低）
xs = np.linspace(0, 1000, 5000)  # 采样点多些曲线更光滑
ys = np.array([my_sound_ds(x) for x in xs])
plt.plot(xs, ys)
plt.xlim(0, 1000)
plt.ylim(-0.1, 0.5)
plt.xlabel("对比音基频 x / Hz")
plt.ylabel("不和谐度（任意单位，相对大小有意义）")
plt.show()
```

## 和弦乐理

### 音程关系

- **两个维度**：**度数**（同度、二度…八度、九度…）看 **谱面上的音名** 距离(如C-G就是五度)；**音数**（含半音个数）决定 **大、小、纯、增、减**。同一度数可以对应不同半音数（如大三度 4 半音、小三度 3 半音）。
- **纯音程（Perfect）**：自然出现在 **一、四、五、八** 度上，无“纯二/三/六/七度”的说法。纯一度、纯八度半音数为 0、12（八度内）；纯四、纯五在十二平均律下分别为 5、7 半音(如C-F/C-G)。**纯音程** 只能 **增、减**，无“大、小”。其中 **增四度** 与 **减五度** 在十二平均律里常是 **同一音高**（6 个半音），但 **度数不同**（四度 vs 五度），在和声进行与解决方向里仍按不同音程处理。
- **三全音（Tritone）**：字面即 **三个全音** 叠起来的距离——一个 **全音** 为 **2 个半音**，故 **3×2 = 6 半音**；在一个八度里 **12 半音取一半** 也是 **6 半音**，所以三全音又常说成把八度 **对分** 的那一档。**三全音 = 6 半音**，与 **增四度**、**减五度** 的 **音宽** 相同（例：C–F♯ 记成增四，C–G♭ 记成减五；十二平均律下两音可同键）。古典和声里它 **极不稳定**，典型如 **属七和弦** 里 **三音到七音** 恰为三全音，倾向 **解决** 到主和弦；分析里说的 **“三全音代理”** 等，指的也是这个 **6 半音** 关系。
- **大、小音程（Major / Minor）**：用于 **二、三、六、七** 度。**大** 比 **小** 多半音（大六度 9 半音 vs 小六度 8 半音等）。大、小音程可再 **增、减** 半音：如 **增三度**（大三度+半音，与纯四度同键）、**减三度**（小三度−半音，与大二度同键）；**减七度**、**增二度** 等在调式与转调分析里常出现。
- **增、减（Augmented / Diminished）**：在纯或大、小音程上 **加/减** 半音得到。**倍增、倍减** 较少见，多见于和声题或变和弦语境。记忆：**增音程** 往往更“宽”，**减音程** 更“窄”。
- **转位（Inversion）**：八度内音程转位规则：**大↔小、增↔减、纯↔纯**；度数相加为 9（如三度↔六度、二度↔七度、四度↔五度）。转位后 **音数之和恒为 12**（一个八度内），用于验算。
- **与和弦的关系**：三和弦由 **根—三、根—五** 决定性质（大三+纯五=大三和弦；小三+纯五=小三和弦）；七和弦还要看 **根—七** 是大七、小七还是减七。**读谱与和声分析** 时，先在脑中拆成 **纵向音程**，再叠成和弦类型。


> [!EXAMPLE]+ 转位例子
>
> **转位与“度数和为 9、半音和为 12”**
> - **大三度** C–E（4 半音）转位成 **小六度** E–C（8 半音）：3+6=9，4+8=12。
> - **小二度** C–D♭（1 半音）转位成 **大七度** D♭–C（11 半音）：2+7=9，1+11=12。
> - **纯四度** C–F（5 半音）转位成 **纯五度** F–C（7 半音）：4+5=9，5+7=12。
> - **增四度** C–F♯（6 半音）转位成 **减五度** F♯–C（6 半音）：增↔减，4+5=9，6+6=12。

> [!EXAMPLE]+ 纵向音程 → 和弦类型（原位、根音在最低声部时）
> - **C 大三和弦** C–E–G：根—三 **大三度**，根—五 **纯五度** → 大三和弦。
> - **A 小三和弦** A–C–E：根—三 **小三度**，根—五 **纯五度** → 小三和弦。
> - **G 属七和弦** G–B–D–F：根—三 **大三度**，根—五 **纯五度**，根—七 **小七度**（G–F）→ 大小七（属七）结构。

### 构成

- **叠置原则**：西方共性写作（Common-practice）里，和弦通常按 **三度叠置** 由低到高排成根音—三音—五音—（七音）—...；根音决定和弦名与级数，叠置层数决定是 **三和弦** 还是 **七和弦** 等。
- **和弦音**：根、三、五、七... 属于该和弦结构本身；声部进行里出现的 **经过、辅助、倚音、换音** 等是旋律层对和弦的修饰，分析时要和和弦音分开。
- **原位与转位**：以 **根音** 为最低音为 **原位**；三和弦中三音、五音在最低音分别为 **第一、第二转位**（数字低音常记 **⁶**、**⁶⁴**）；七和弦另有 **第三转位**（如 **⁴²**）。转位改变低音线条与解决倾向，**不改变** 和弦的根音与功能归属（仍以根音定名）。
- **声部与排列**：同一和弦可 **密集排列** 或 **开放排列**；四部和声写作里需注意 **重复音规则**（通常重复根或五，属和弦解决等情形另有惯例）。与前文物理学部分衔接：和弦“构成”回答的是 **同时选了哪些音级**；不和谐感则来自各音分音列之间的相互作用。


> [!EXAMPLE]+ 构成：例子
>
> **三度叠置**：**C 大三和弦** 由低到高为 **C–E–G**（先大三度 C–E，再叠小三度 E–G，整体仍是根—三—五）。若在 **C–E–G–B** 上再加三度，得到 **C 大七**（根—三—五—七）。
> **和弦音 vs 装饰音**：若当前和弦是 **C 大三**（C–E–G），旋律里短暂出现的 **F** 可能是 **经过音** 去往 **E**，分析时 **和弦仍是 C**，F 不记入和弦结构。
> **原位与转位（同一和弦、不同低音）**：**C–E–G** 原位；**E–G–C**（三音在最低）为 **第一转位**，常分析为 **I⁶**；**G–C–E** 为 **第二转位**，常记 **I⁶⁴**。根音仍是 **C**，和弦名仍是 **C 大三**。
> **密集 / 开放**：四个声部弹同一 **C 和弦**，三音、五音与根音 **挤在同一八度** 内偏 **密集**；把五音或三音 **翻到更高八度** 则 **更开放**，和弦种类不变。

### 类型

以下类型均指 **原位**、从 **根音** 往上看 **根—三—五—（七）** 的 **音程叠置**；转位只改变低音，不改变类型命名。

#### 三和弦
（Triads）

- **大三和弦（Major triad）**：根—三 **大三度**，根—五 **纯五度**（三音与五音之间为 **小三度**）。色彩偏 **明朗、稳定**；自然大调 **I、IV、V** 级上的三和弦均为大三（在调式内讨论时）。
- **小三和弦（Minor triad）**：根—三 **小三度**，根—五 **纯五度**（三—五为 **大三度**）。色彩偏 **暗、柔**；自然大调 **ii、iii、vi** 级为 **小三和弦**（级数随调而变）。
- **减三和弦（Diminished triad）**：根—三 **小三度**，根—五 **减五度**（三—五仍为 **小三度**）。**极不稳定**，常向相邻和弦解决；自然大调 **vii°**（导三和弦）即减三结构。
- **增三和弦（Augmented triad）**：根—三 **大三度**，根—五 **增五度**（三—五为 **大三度**）。对称、暧昧，古典中不如前三种常见，多见于 **经过、半音和声** 或浪漫派语汇。

<br>

#### 七和弦
（Seventh chords）

在 **三和弦** 上再叠 **七度音**（根—七为 **七度音程**）。分类看 **根—三、根—五、根—七** 三个纵向音程（原位）。

- **大小七和弦**（**Major-minor 7th**，常称 **属七**、**Dominant 7th**）：根—三 **大三度**，根—五 **纯五度**，根—七 **小七度**（即 **大三和弦 + 小七度**）。**紧张度强**，典型 **属功能**（如调内 **V⁷**），强倾向解决到主。
- **大七和弦**（**Major 7th**，**MM7** / **maj⁷**）：根—三 **大三度**，根—五 **纯五度**，根—七 **大七度**（**大三 + 大七**）。色彩 **亮而厚**，大调 **Imaj⁷、IVmaj⁷** 等常见；与属七相比 **根—七更宽**（大七 vs 小七）。
- **小七和弦**（**Minor 7th**，**mm7**）：根—三 **小三度**，根—五 **纯五度**，根—七 **小七度**（**小三 + 小七**）。色彩 **柔、略飘**，如 **ii⁷**（下属区域）、**vi⁷**、**iii⁷** 等（级数依调）。
- **半减七和弦**（**Half-diminished 7th**，**ø⁷** / **m7♭5**）：根—三 **小三度**，根—五 **减五度**，根—七 **小七度**（**减三和弦 + 小七度**）。**导音色彩**强；大调 **viiø⁷**（导上的半减七）、小调 **iiø⁷**（二级上的半减七，如 **iiø⁶⁵–V** 进行）等均为常见身份。
- **减减七和弦**（**Diminished 7th**，**°⁷** / **dim⁷**）：根—三 **小三度**，根—五 **减五度**，根—七 **减七度**（**减三 + 减七**；四音由 **小三度** 连续叠成，**结构全对称**）。**极紧张**，常作 **经过、等音转调** 枢纽；和声小调 **#vii°⁷**（导七）是典型环境之一。

**小大七和弦**（**Minor-major 7th**）：根—三 **小三度**，根—五 **纯五度**，根—七 **大七度**（小三和弦 + 大七度）。爵士记 **mMaj7**，古典中相对少，可见于 **旋律小调上行** 等语境；需要时单独认结构即可。


<br>

#### 和弦功能

【主（T）、下属（S）、属（D）】

除“和弦由哪些三度叠成”之外，共性期分析还要问：该和弦在 **当前调** 里 **扮演什么角色**(**稳定还是紧张**、**倾向解决到哪**)。德语系理论常简记为 **T / S / D**（英：Tonic / Subdominant / Dominant）。下面以 **C 大调** 为例（换调时 **罗马数字级数不变、音名整体平移**）。

- **主功能（T，Tonic）**  
  - **核心**是 **I**（主和弦）。音乐 **开始、结束** 最常落在 **I** 或 **I⁶**，听觉上是 **“家”**、调性锚点。  
  - **iii、vi** 常与主区 **共享音级**（如 **C 大调** 里 **E–G–B、A–C–E** 都含主和弦音），可作 **主功能的替代或过渡**，色彩更柔或更暗，但 **终止感** 一般不如 **I** 干脆。  
  - **一句话**：**相对最稳**；全曲的 **参照中心**。

- **属功能（D，Dominant）**  
  - **核心**是 **V**；**vii°**（减三和弦在导音上）与 **导七** 等也归 **属侧**，因都强调 **导音 B→C**（向主音的 **半音解决**）与 **属音 G→C** 的倾向。  
  - **V⁷、V⁶⁵** 等属七类 **张力更强**（三全音要“掰开”），**最典型** 的解决是落回 **I**（**正格终止 V–I / V⁷–I**）。  
  - **一句话**：**最指向主**、**最紧张** 的一极；和声进行里 **“收紧”** 的那一步往往在这里。

- **下属功能（S，Subdominant）**  
  - **核心**是 **IV**；**ii**（或 **ii⁶、ii⁷**）也常在 **下属区域**，负责 **主与属之间的“铺开”**：音阶上在 **主音之下** 一侧（如 **F、D** 相对 **C**），色彩比 **V** **更宽、更柔**，又比 **I** **更动**。  
  - **常见走向**：**IV–V–I、ii–V–I、ii⁶–V⁷–I**——先经 **S** **垫高张力**，再由 **D** **一把收进主**；也可 **IV–I**（**变格终止**）**不经属** 直接回主，色彩不同。  
  - **与属的区分**：**S** 多是 **铺垫、展开**（“要离开家出去走走”）；**D** 是 **对准家门钥匙孔**（“非解决不可”）。同一 **小七和弦**：**ii⁷** 多在 **S**，**V⁷** 在 **D**——**结构可相似，功能标签不同**。

- **典型骨架**：**T →（S）→ D → T**。不必每句都凑齐三步，但 **D → T**（**属→主**）是最基本的 **正格** 咬合；**S → D → T** 是教科书式的 **完全进行** 之一。

- **扩展**：九、十一、十三和弦及 **变和弦**（增六、那不勒斯等）在古典晚期与浪漫派大量出现；**副属、借用、转调** 会让和弦暂时脱离本调 **T/S/D** 标签，记谱与分析符号依教材与曲目另学。


> [!EXAMPLE]+ 功能：主 / 下属 / 属（以 C 大调为例）
>
> **主（T）**：**I** = **C–E–G**；**vi** = **A–C–E**；**iii** = **E–G–B**（后两者常作主区色彩变化，终止仍以 **I** 最稳）。
>
> **下属（S）**：**IV** = **F–A–C**；**ii** = **D–F–A**（或 **ii⁶**、**ii⁷**）；常见片段 **IV–V–I**、**ii–V–I**。
>
> **属（D）**：**V** = **G–B–D**；**V⁷** = **G–B–D–F**；**vii°** = **B–D–F**；典型解决 **V(⁷) → I**。
>
> **串起来**：**I–IV–V–I**（**C–F–G–C**）= **T–S–D–T**；**ii–V–I**（**D–F–A → G–B–D → C–E–G**，中间也常接 **V⁷**）= **S–D–T**。


> [!EXAMPLE]+ 类型：例子（一类一行）
>
> **三和弦**
> - **大三**：**C–E–G**（大三+纯五）。
> - **小三**：**A–C–E**（小三+纯五）；**C–E♭–G**。
> - **减三**：**B–D–F**（小三+减五）；**C–E♭–G♭**。
> - **增三**：**C–E–G♯**（大三+增五）。
>
> **七和弦**
> - **大小七（属七）**：**G–B–D–F**（大+纯五+小七）；**C–E–G–B♭**。
> - **大七**：**C–E–G–B**（大+纯五+大七）。
> - **小七**：**A–C–E–G**（小+纯五+小七）；**C–E♭–G–B♭**。
> - **半减七**：**B–D–F–A**（小+减五+小七）；**C–E♭–G♭–B♭**。
> - **减减七**：**B–D–F–A♭**（小+减五+减七；平均律下 **A♭** 与 **G♯** 等音可互换记谱）；**C–E♭–G♭–A**（根上减七度理论为 **B 重降**，十二平均律下与 **A** 等音）。
> - **小大七**（补充）：**A–C–E–G♯**（小+纯五+大七）；**C–E♭–G–B**。
>
> **类型 vs 功能**：**G–B–D–F** 在 **C 大调** 为 **V⁷**；在 **G 大调** 中同一叠置写在 **D–F♯–A–C** 上仍为 **V⁷**——结构名相同，**调内级数** 随调而变。

### 标识

- **字母和弦（流行、爵士、歌曲谱常用）**  
  写成 **C、Cm、C°、C+、C7、Cm7、Cmaj7** 等，表示 **固定音高上的根音 + 和弦类型**：**C** 即“以 **C** 为根”、**m** 为小三、**7** 为属七……**不自带调号信息**——同一串字母在不同调的歌里，含义都是“这些音名叠成的和弦”。**换调** 时，若和声进行不变，要 **整套改字母**（例如原调 **C–F–G–C** 移到 **G** 大调常写成 **G–C–D–G**）。

- **罗马数字（古典和声分析、调性写作课的主标记）**  
  写成 **I、ii、iii、IV、V、vi、vii°** 等，表示 **当前调** 里的 **级数 + 和弦性质**：**大写** 多为 **大三和弦**（或以大三和弦为骨架的七和弦），**小写** 多为 **小三**；**°** 为减三、**⁷** 为七和弦等（细则随教本）。**同一罗马数字** 在 **不同调** 里对应 **不同音名**——例如 **V** 在 **C 大调** 是 **G**，在 **G 大调** 是 **D**，但 **在各自调里都是“属和弦”那一级”**。

- **两套符号怎么对应（核心）**  
  | | **字母标记** | **罗马数字** |
  |---|---|---|
  | **回答的问题** | 根音 **绝对** 是哪个音名、叠置类型是什么 | 在 **本调** 里是 **第几级**、与 **主音** 的关系 |
  | **是否依赖调** | 否（只依赖“根音 + 类型”） | **是**（必须先明确 **是什么调**） |
  | **典型用途** | 移调乐器、乐队谱、流行 lead sheet | 和声分析、作曲课、总谱读法中的 **功能与进行** |

  **同一条进行，两种写法**：在 **C 大调** 里 **C – F – G – C** 与 **I – IV – V – I** 描述的是 **同一串和弦**；把调换到 **G 大调**，字母变成 **G – C – D – G**，罗马数字 **仍是** **I – IV – V – I**（级数与功能不变，音名整体平移）。

- **转位标记**：如 **I⁶**（一和弦第一转位）、**V⁶⁵**（属七及其转位的一种记法，以所用体系为准）、**I⁶⁴** 等；不同教本对 **六四和弦**（经过、倚音、终止式）细分不一，作业以课程规定为准。
- **数字低音 / Figured bass**：巴洛克—古典早期 **通奏低音** 乐谱上的 **⁶、⁶⁴、⁷、⁴³** 等，表示 **相对最低音** 应构成的音程；读巴赫、亨德尔钢琴声部或总谱时与 **现代和弦缩写** 是两套符号，需要单独习惯。
- **功能标记**：字母 **T、S、D** 表示 **主 / 下属 / 属** 功能（见上文 **“和弦功能”** 一节）。书写时常标在 **和弦进行** 下方或行间，如 **T | S | D | T**，概括 **I–IV–V–I** 等。**D–T** 特指 **属→主** 的 **正格** 收束；**S–D–T** 指 **含下属预备** 再进属、回主。**与罗马数字并用** 时：罗马数字说 **是哪一级**，**T/S/D** 说 **在调性张力里的角色**。

> [!EXAMPLE]+ 标识：音名字母 ↔ 罗马数字（对应关系）
>
> **C 大调内（同一调里对照）**
> - **I** ↔ **C**（大三）　**IV** ↔ **F**（大三）　**V** ↔ **G**（大三）　**V⁷** ↔ **G7**
> - **ii** ↔ **Dm**　**iii** ↔ **Em**　**vi** ↔ **Am**　**vii°** ↔ **B°**（减三）
>
> **换调：罗马数字可不变，字母要整套换**  
> - **I – IV – V – I**：在 **C 大调** = **C – F – G – C**；在 **G 大调** = **G – C – D – G**（级数相同，根音随调平移）。
> - **ii – V – I**：在 **C 大调** = **Dm – G – C**；在 **G 大调** = **Am – D – G**。
>
> **字母-only 与罗马-only**  
> - **G7 → C**：字母告诉你 **属七 → 主** 的音响；在 **C 大调** 里分析常写成 **V⁷ → I**，在 **G 大调** 里同一对音响是 **D7 → G**（仍是 **V⁷ → I**）。**罗马数字抓“在调里的角色”，字母抓“具体音名”。**
>
> **转位**：**I⁶** 在 C 大调里常指 **C 大三第一转位**（低音 **E**，亦有人记 **C/E**）；**I⁶⁴** 常指 **第二转位**（低音 **G**）。**V⁶⁵** 等以 **教本** 为准。
>
> **数字低音**：通奏低音 **⁶、⁶⁴、⁷** 与 **字母/罗马** 是第三套符号，读早期乐谱时再专门对。
>
> **功能标记**：**I – IV – V – I** 可标 **T – S – D – T**；**ii – V – I** 可标 **S – D – T**。**IV – I** 为 **S – T**（变格）。
