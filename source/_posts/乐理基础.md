## 声音物理

### 声音的产生

- 声音产生的三个条件：声源(振动的琴弦)、介质(空气)、受体(人耳)


| | 声音的频率 | 声音的振幅 |
| :--- | :--- | :--- |
| **定义** | 每秒钟声波振动的次数 | 声音的音量，或者说响度、功率、能量 |
| **听力范围** | 20 赫兹 — 20,000 赫兹（20 Hz — 20,000 Hz） | 20 微帕 — 200,000,000 微帕（20 μPa — 200,000,000 μPa） |
| **人的感受** | 频率低，听起来比较低沉、飘渺；<br>频率高，听起来比较高亢、激越 | 振幅低，代表声音小；<br>振幅高，代表声音大 |

【**自然界不存在单一频率的声音**，任何一种声音都是各种频率声波的独特组合。单一频率声音只有实验室里有。】

- 噪音： 杂乱无章的频率与音量
- 乐音：
    - 清晰的音高感：对音调高低有明确的感觉
    - 清晰的音色感：可以分辨由哪种乐器发出的声音

### 五线谱
认识五线谱是从小学钢琴的基本功，不写了。

#### 谱号

<img src="https://raw.githubusercontent.com/YukinoshitaSherry/qycf_picbed/main/img/20260411171035152.png"/><br>

**谱号与中央 C（便于对照）**：高音谱号下加一线、中音谱号正中间一线、低音谱号上加一线，通常都对准 **中央 C**（钢琴白键里大致在中间的那一组 C）。读谱时先把谱号对应的一线一音立住，再向上下数。

**常见用法（弦乐空弦便于记音高）**：

| 谱号 | 典型乐器 | 弦乐空弦（由低到高，四根弦） |
| :--- | :--- | :--- |
| 高音（G） | 小提琴、长笛、双簧管、小号、钢琴右手等 | **小提琴**：G、D、A、E |
| 中音（C，中音谱号） | **中提琴**、长号高音区记谱等 | **中提琴**：C、G、D、A（整体比小提琴低五度） |
| 次中音（C，第四线为中央 C） | 大提琴高音区、大管、长号等 | **大提琴**：空弦为 C、G、D、A（记谱常在低音谱号与次中音谱号之间换） |
| 低音（F） | 大提琴低音区、低音提琴、巴松、长号低音、钢琴左手等 | **低音提琴**：E、A、D、G（定弦为四度；**实际音高比记谱低八度**，总谱里常带八度说明） |

钢琴双手：**右手多为高音谱号、左手多为低音谱号**；竖琴类似。铜管、木管按音域分谱，同一乐器也可能换谱号（例如大提琴、长号）。

### 总谱


**总谱**（Full score）是把 **同一时间轴上** 各声部、各乐器的谱表 **纵向叠在一起** 印成的总册；指挥、作曲分析、配器学习都以它为“全团地图”。读的时候心里要有两件事：**横轴是时间**，**纵轴是音色分层**。

**读法**：

- **从上到下**：习惯上常按 **木管 → 铜管 → 打击乐与色彩乐器（竖琴、钢琴等）→ 弦乐** 排列（具体版本会微调，但弦乐多在 **最下方**）。越靠上的声部，在总谱里位置越“高”，但 **音区未必更高**，只是排版惯例；真实高低仍以各声部自己的谱号与八度记号为准。
- **竖向对齐**：同一条 **纵切线** 上的音符，表示 **同时发声**（或同一拍点上的起音）。读总谱就是在脑子里做 **纵向和声切片**，再横向跟着小节线推进。
- **与分谱**：每位演奏者手里是 **自己的一行或一组**（分谱）；总谱是指挥与作曲家的视角，看的是织体与配器关系。

## 五度圈

把十二个半音按 **连续纯五度**（或等价地，反方向连续纯四度）排成环，就得到五度圈。它把“音程的物理来源”和“调性音乐的语法”联系在同一张图上。

<img src="https://raw.githubusercontent.com/YukinoshitaSherry/qycf_picbed/main/img/20260411170115057.png"/><br>

- **与泛音、整数比的同一根线**：纯五度来自低序号谐波比（近似 $3:2$），所以在泛音列里出现得早、耳朵觉得“稳”。前文“公倍数泛音很和谐”与“五度叠置生成调域”是同一套和谐观：五度圈不是凭空记号，而是把这条物理直觉组织成 **调号与远近关系** 的工具。
- **与十二平均律的闭合**：十二平均律里，纯五度是 $2^{7/12}$ 倍频；沿五度方向走 12 步会 **回到同一音名的类**，形成一个闭合环（与“把圆周十二等分”是同构想法，只是音乐上用的是对数音高）。
- **调号的辨析**：从 C 大调出发，沿五度圈 **顺时针** 每多一站，调号里就多 **一个升号**；升号出现的 **先后顺序** 固定为 **F→C→G→D→A→E→B**。**逆时针** 则多 **降号**；降号顺序为 **B→E→A→D→G→C→F**（与上一串 **顺序相反**）。因此圈上的位置直接对应有几个升号或几个降号。顺时针常视为“升号方向”、逆时针为“降号方向”；相邻两调在五度圈上相邻，共享和弦多、转调自然，故称 **近关系调**。属（dominant）相对主音恰在五度之上，所以 **属—主** 进行在五度圈上就是“向主音收束”的一步——和声学里的功能进行，与几何上的圈是同一结构的不同说法。

## 中古调式

在 **自然七声音列**（本文前述十二平均律中取定的七个音级）上，把 **不同音级当作主音（tonic）** 建立中心，就得到七种 **中古调式**（教会调式）。

<img src="https://raw.githubusercontent.com/YukinoshitaSherry/qycf_picbed/main/img/20260411165725261.png"/><br>

以 C 大调自然音列为参照 (任何基音都可以参照，写作 C Ionian、D Dorian...)：
<img src="https://raw.githubusercontent.com/YukinoshitaSherry/qycf_picbed/main/img/20260411165358616.png"/><br>

- 规律：以 C 大调自然音列为参照，最明亮的是利底亚，在自然大调基础上#F，之后依次按照五度圈顺时针顺序每次降一个音,按明亮到黑暗的顺序为：（利底亚-伊奥尼亚-混合利底亚-多利亚-爱奥尼亚-弗吉利亚-洛克利亚）。

| 名称 | 常见对应 | 相对自然大调（Ionian）的音程特征（抓耳的） |
| :--- | :--- | :--- |
| Ionian | 自然大调 | 大调七声；明亮、稳定 |
| Dorian | — | 小调色彩但 **大六度**（相对自然小调），常听来既阴又亮 |
| Phrygian | — | **小二度** 主音上方，西班牙/金属味常见 |
| Lydian | — | **增四度** 相对大调，飘、梦幻、现代配乐常用 |
| Mixolydian | — | **小七度** 相对大调，属七色彩、民谣/摇滚常见 |
| Aeolian | 自然小调 | 小调七声；忧郁、叙事 |
| Locrian | 极少作主调 | **减五度** 相对小调，极不稳定，更多用于局部色彩或理论完整 |

**与大小调的关系**：今天最常用的是 **Ionian / Aeolian**（自然大、自然小）；其余调式可看作在相同自然音集合里 **换主音**，于是出现特征音程。爵士与当代音乐里常直接说“在某和弦上弹某调式音阶”，本质也是 **把和弦当作临时的泛音/功能中心** 来选音——与中古调式“换主音”同一逻辑，和声语汇更复杂而已。


## 五声调式

中国传统常用的 **宫、商、角、徵、羽** 是在一个八度内选定的五个核心音级；在此基础上可加入 **偏音**（如清角、变宫等）发展为六声、七声。


**与中古调式对比** ：
- 联系： 都要回答 **以哪一级音为中心**、**音与音的相对距离如何塑造旋律色彩**；都可以在 **不更换整套音名体系** 的前提下，通过 **换中心音或换落音习惯** 得到不同调式色彩，因而都适合做 **调式分析**，而不是只数音符个数。

-区别：* *音级骨架**。中古调式是在 **七声自然音列** 上换主音，默认始终带着大调式或小调式那一套 **半音与全音的间隔模式**。五声传统以 **五声为骨**，七声往往是 **五声加偏音** 的结果，旋律上常见 **强调级进中的大三度、四度跳进** 与 **规避某类半音邻音** 的习惯；中古调式以 **七声齐备** 为常态，Locrian 等模式里 **减五度、小二度** 等色彩来自 **七声结构本身**，与五声骨架上 **偶现偏音** 的听感策略并不相同。
